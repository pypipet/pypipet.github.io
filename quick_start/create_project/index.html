    Start a project

    Create new project¶

        mkdir my_project_folder
    cd my_project_folder
    pypipet new project_name
    In current folder

    .
+-- setting.yaml
+-- bundle
|   +-- db_setting.yaml
|   +-- file_template.yaml
|   +-- wc_field_mapping.yaml
    +-- spf_field_mapping.yaml

    Note: run pypipet in the project folder (where the configuration filsetting.yaml is)

    Customize setting files¶

    • setting.yaml project setting
    • bundle/db_setting.yaml database auth and schema
    • bundle/file_template.yaml format of template files
    • bundle/wc_field_mapping.yaml woocommerce API field mapping
    • bundle/spf_field_mapping.yaml shopify API field mapping

    Here are examples of setting files of project configuration.

    Here are the references to set up shop API.

    Initialize Project¶

    pypipet initialize

    database tables are automatically set up accordiing to bundle/database_setting.yaml

    see database schema

    For terms used in Pypipet, please check Terms.

    Import Product Catalog¶

    option 1: import from file¶

    1. generate template

      pypipet template frontshop
pypipet template supplier
pypipet template tax
pypipet template product

      *notes about template files

      • The template are csv files. The column name format is Tablename.Columnname.
      • For importing catalog only, ignore the columns with name start with Destination

    2. fill the data in files (csv files)

    3. load the catalog into database

      pypipet static add --target supplier -f supplier_template.csv
pypipet static add --target supplier -f frontshop_template.csv
pypipet static add --target supplier -f tax_template.csv
pypipet catalog file --shop [your_shop_name] -f product_template.csv

      refer your_shop_name to shop name in setting.yaml

    option 2: import from shop¶

        pypipet catalog shop --shop [your_shop_name]

    update product information¶

    show product infomation¶

    pypipet product  show  --shop  [your_shop_name]  --sku  [your_product_sku]

    edit product information¶

    pypipet product  edit  --shop  [your_shop_name]  -f  product_update_template.csv

    launch a product to a frontshop¶

    pypipet product  launch  --shop  [your_shop_name]  --sku  [your_product_sku] --price [your_lauch_price]

    inventory management¶

    pypipet inventory edit --shop [your_shop_name] -f inventory_template.csv

    Order Management¶

    synchronize orders from frontshop to database¶

    pypipet order  sync  --shop  home_site

    show order information¶

    pypipet order  show  --shop  [your_shop_name]  --id  [your_order_id]

    your_order_id use frontshop order id

    update order status¶

    pypipet order  status  --shop  [your_shop_name]  --id  [your_order_id] --manual refunded

    Add tracking for shipment¶

    add tracking for an order¶

    pypipet fulfillment  add  --shop  [your_shop_name]  --order-id  [your_order_id]  --provider  [your_service_provider]  --tracking  [your_tracking_id]

    your_order_id use frontshop order id

    your_service_provider e.g., canada_post,usps, ups, fedex, dhl

    bulk add tracking for orders¶

    pypipet fulfillment  add  --shop  [your_shop_name] -f fulfillment_template.csv

    show tracking information of an order¶

    pypipet fulfillment  show  --shop  [your_shop_name]  --order-id  [your_order_id]

    Import static data to database¶

    shops¶

    pypipet static  add  --target  front_shop  -f  frontshop_template.csv

    supplier¶

    pypipet static  add  --target  supplier  -f  supplier_template.csv

    tax¶

    pypipet static  add  --target  tax  -f  tax_template.csv

    category¶

    pypipet static  add  --target  category  -f  category_template.csv
